This skill performs comprehensive code reviews focusing on:
- Bug detection
- Security vulnerabilities
- Performance optimization
- Best practices adherence
- Code readability

## Review checklist

1. **Correctness**: Does the code do what it's supposed to?
2. **Security**: Are there any security vulnerabilities?
3. **Performance**: Can the code be optimized?
4. **Style**: Does it follow language conventions?
5. **Readability**: Is the code easy to understand?
6. **Testing**: Are there adequate tests?

## Output format

Provide a structured review with:
- Summary of findings
- Critical issues (must fix)
- Suggestions for improvement
- Positive aspects of the code
